Stochastic Process Basics for Brownian Motion

  1. Foundation: Random Variables in Continuous Time

A stochastic process is a collection of random variables indexed by time: {X(t):t∈T}\{X(t) : t \in T\}{X(t):t∈T}. Think of it as a random function of time, where at each moment ttt, X(t)X(t)X(t) is a random variable.

For discrete processes you know well (like random walks), we have X0,X1,X2,…X_0, X_1, X_2, \ldotsX0​,X1​,X2​,… In continuous time, we have X(t)X(t)X(t) for all t∈[0,∞)t \in [0, \infty)t∈[0,∞).

  1. Brownian Motion (Wiener Process)

Definition: A process B(t)B(t)B(t) is Brownian motion if:

  1. B(0)=0B(0) = 0B(0)=0

  2. Independent increments: B(t2)−B(t1)B(t_2) - B(t_1)B(t2​)−B(t1​) is independent of B(t4)−B(t3)B(t_4) - B(t_3)B(t4​)−B(t3​) for non-overlapping intervals

  3. Stationary increments: B(t+s)−B(t)∼N(0,s)B(t+s) - B(t) \sim \mathcal{N}(0, s)B(t+s)−B(t)∼N(0,s) for all t,s>0t, s > 0t,s>0

  4. Continuous paths (but nowhere differentiable!)

Key insight: Brownian motion is the continuous-time limit of a random walk with step size → 0 and step frequency → ∞.

  1. The Problem with Traditional Calculus

In normal calculus, we'd write:

dXdt=f(X,t) ⟹ dX=f(X,t)dt\frac{dX}{dt} = f(X, t) \implies dX = f(X, t)dtdtdX​=f(X,t)⟹dX=f(X,t)dt

But Brownian motion B(t)B(t)B(t) is nowhere differentiable, so dB/dtdB/dtdB/dt doesn't exist in the classical sense. We need a new framework.

3.5. Why BM is Not Differentiable But We Can Still Use dB

The Non-Differentiability Proof (Intuition)

Why BM isn't differentiable: Consider the derivative definition:

B′(t)=lim⁡h→0B(t+h)−B(t)hB'(t) = \lim_{h \to 0} \frac{B(t+h) - B(t)}{h}B′(t)=h→0lim​hB(t+h)−B(t)​

For this limit to exist, we need (B(t+h)−B(t))/h(B(t+h) - B(t))/h(B(t+h)−B(t))/h to converge as h→0h \to 0h→0. But:

  • B(t+h)−B(t)∼N(0,h)B(t+h) - B(t) \sim \mathcal{N}(0, h)B(t+h)−B(t)∼N(0,h) for any h>0h > 0h>0

  • So B(t+h)−B(t)h∼N(0,hh2)=N(0,1/h)\frac{B(t+h) - B(t)}{h} \sim \mathcal{N}\left(0, \frac{h}{h^2}\right) = \mathcal{N}(0, 1/h)hB(t+h)−B(t)​∼N(0,h2h​)=N(0,1/h) for any h>0h > 0h>0

    • Note: If Z∼N(0,σ2)Z \sim \mathcal{N}(0, \sigma^2)Z∼N(0,σ2), then Z/c∼N(0,σ2/c2)Z/c \sim \mathcal{N}(0, \sigma^2/c^2)Z/c∼N(0,σ2/c2)

As h→0h \to 0h→0, the variance 1/h→∞1/h \to \infty1/h→∞. The ratio doesn't converge; it explodes! Almost surely, for any ttt, the derivative B′(t)B'(t)B′(t) doesn't exist.

Geometric intuition: BM has infinite "wiggling" at every scale. Zoom in on any point, and you see more jagged variation, never smoothing out to a tangent line.

So Why Can We Write dB?

Key distinction: dBdBdB is NOT (dB/dt)⋅dt(dB/dt) \cdot dt(dB/dt)⋅dt. It's a fundamentally different object.

Is dBdBdB a differential? Not in the classical sense. In traditional calculus, differentials are related to derivatives: dy=f′(x)dxdy = f'(x)dxdy=f′(x)dx where f′f'f′ exists. Here, dBdBdB is a stochastic differential - a new type of object defined through the integration theory itself, not through differentiation.

What dB means:

dB=dB(t)=B(t+dt)−B(t)dB = dB(t) = B(t+dt) - B(t)dB=dB(t)=B(t+dt)−B(t)

This is an infinitesimal random increment, not a derivative times dtdtdt. Think of it as:

  • dBdBdB: a random variable with distribution N(0,dt)\mathcal{N}(0, dt)N(0,dt)

  • Not decomposable into (dB/dt)⋅dt(dB/dt) \cdot dt(dB/dt)⋅dt because dB/dtdB/dtdB/dt doesn't exist

  • Best thought of as a "random infinitesimal" - a fundamental building block, not derived from anything else

The Mathematical Framework: Stochastic Integration

Traditional calculus: Integration is defined via derivatives (Fundamental Theorem of Calculus)

Stochastic calculus: Integration is defined directly via limits of sums:

∫0Tf(t)dB(t)=lim⁡n→∞∑if(ti)(B(ti+1)−B(ti))\int_0^T f(t)dB(t) = \lim_{n \to \infty} \sum_{i} f(t_i)(B(t_{i+1}) - B(t_i))∫0T​f(t)dB(t)=n→∞lim​i∑​f(ti​)(B(ti+1​)−B(ti​))

This integral is well-defined even though B(t)B(t)B(t) has no derivative! We're:

  1. Partitioning the interval [0,T][0, T][0,T]

  2. Taking increments ΔBi=B(ti+1)−B(ti)\Delta B_i = B(t_{i+1}) - B(t_i)ΔBi​=B(ti+1​)−B(ti​)

  3. Summing f(ti)⋅ΔBif(t_i) \cdot \Delta B_if(ti​)⋅ΔBi​

  4. Taking limit as partition gets finer

Key insight: We never need derivatives. We only need:

  • The path B(t)B(t)B(t) itself (which is continuous)

  • The increments B(t2)−B(t1)B(t_2) - B(t_1)B(t2​)−B(t1​) (which are well-defined)

Differential Notation is Shorthand

When we write:

dX=μ dt+σ dBdX = \mu \, dt + \sigma \, dBdX=μdt+σdB

We really mean the integral equation:

X(t)−X(0)=∫0tμ(s)ds+∫0tσ(s)dB(s)X(t) - X(0) = \int_0^t \mu(s)ds + \int_0^t \sigma(s)dB(s)X(t)−X(0)=∫0t​μ(s)ds+∫0t​σ(s)dB(s)

X(t+dt)−X(t)≈μ(t)dt+σ(t)dB(t)X(t+dt) - X(t) \approx \mu(t)dt + \sigma(t)dB(t)X(t+dt)−X(t)≈μ(t)dt+σ(t)dB(t)

Analogy to discrete math:

  • In difference equations: Xn+1−Xn=f(Xn)X_{n+1} - X_n = f(X_n)Xn+1​−Xn​=f(Xn​)

  • We might write: ΔXn=f(Xn)\Delta X_n = f(X_n)ΔXn​=f(Xn​)

We're not claiming dX/dtdX/dtdX/dt exists; we're describing how XXX increments over infinitesimal time intervals.

Why This Works

Regularity requirements:

  1. B(t)B(t)B(t) is continuous (can take limits)

  2. B(t)B(t)B(t) has bounded variation on finite intervals (integration works)

  3. Increments are independent and Gaussian (can compute expectations)

Even without differentiability, these properties suffice to build a rigorous integration theory (Itô, 1944).

Bottom line:

  • dB/dtdB/dtdB/dt doesn't exist ✗

  • dBdBdB exists as a random infinitesimal ✓

  • ∫f dB\int f \, dB∫fdB exists as a well-defined limit ✓

  • Differential notation dBdBdB is shorthand for integration, not differentiation

  1. Stochastic Differentials: The Intuition

The Differential Notation

When we write dB(t)dB(t)dB(t) (or dW(t)dW(t)dW(t)), we mean:

dB(t)=B(t+dt)−B(t)dB(t) = B(t + dt) - B(t)dB(t)=B(t+dt)−B(t)

This is an infinitesimal increment of Brownian motion. Key properties:

  • E[dB(t)]=0\mathbb{E}[dB(t)] = 0E[dB(t)]=0 (zero mean)

  • E[dB(t)2]=dt\mathbb{E}[dB(t)^2] = dtE[dB(t)2]=dt (variance grows linearly with time)

  • dB(t)∼N(0,dt)dB(t) \sim \mathcal{N}(0, dt)dB(t)∼N(0,dt)

The Scaling Mystery

Here's the non-intuitive part from regular calculus:

dB(t)=O(dt)NOT O(dt)dB(t) = O(\sqrt{dt}) \quad \text{NOT } O(dt)dB(t)=O(dt​)NOT O(dt)

This is because variance scales with dtdtdt, so standard deviation scales with dt\sqrt{dt}dt​.

Note:

  • Linear variance growth: Even though dtdtdt is small, the linearity matters - it tells us how uncertainty accumulates over finite time (grows as T\sqrt{T}T​, not TTT)

  • The dt\sqrt{dt}dt​ scaling - why it's crucial:

    • It makes (dB)2=O(dt)(dB)^2=O(dt)(dB)2=O(dt) non-negligible (the foundation of Itô calculus).

    • Diffusion dominates drift at small time scales

    • Random accumulation is sublinear (T\sqrt{T}T​, not TTT)

    • Forces second-order terms in the chain rule

  • drunk-walking analogy: - after NNN random steps, you're N\sqrt{N}N​ blocks away, not NNN blocks. This captures why the t\sqrt{t}t​ scaling is fundamental to understanding random motion!

Drunk-walking analogy: After NNN random steps (each step ±1 block with equal probability), you're typically N\sqrt{N}N​ blocks away from where you started, not NNN blocks away. This N\sqrt{N}N​ scaling captures the "inefficiency" of random exploration - you don't make linear progress because you keep randomly changing direction. The same principle applies to Brownian motion: over time ttt, the typical displacement is O(t)O(\sqrt{t})O(t​), not O(t)O(t)O(t).

  1. Stochastic Differential Equations (SDEs)

A typical SDE looks like:

dX(t)=μ(X,t)dt+σ(X,t)dB(t)dX(t) = \mu(X, t)dt + \sigma(X, t)dB(t)dX(t)=μ(X,t)dt+σ(X,t)dB(t)

Interpretation:

  • μ(X,t)dt\mu(X, t)dtμ(X,t)dt: deterministic drift (like traditional calculus)

  • σ(X,t)dB(t)\sigma(X, t)dB(t)σ(X,t)dB(t): random diffusion (the new part)

Understanding μ\muμ (Drift): Is it a Rate or Percentage?

Short answer: μ\muμ is an instantaneous rate of change with units of [value/time].

Intuitive breakdown:

  • In dX=μ dtdX = \mu \, dtdX=μdt, we have: dXdt=μ\frac{dX}{dt} = \mudtdX​=μ

  • So μ\muμ is literally "how fast XXX changes per unit time" (deterministically)

  • Units: If XXX is dollars and ttt is years, then μ\muμ has units dollars/year

When is μ\muμ a percentage? In finance, for geometric Brownian motion:

dS=μS dt+σS dBdS = \mu S \, dt + \sigma S \, dBdS=μSdt+σSdB

Here:

  • μ\muμ has units [1/time] - it's a rate of return (e.g., 0.05/year = 5% per year)

  • The equation says: "stock price changes by μ\muμ percent per unit time (on average)"

  • Dividing both sides by SSS: dSS=μ dt+σ dB\frac{dS}{S} = \mu \, dt + \sigma \, dBSdS​=μdt+σdB ← log-return!

Key distinction:

  • Arithmetic BM: dX=μ dt+σ dBdX = \mu \, dt + \sigma \, dBdX=μdt+σdB → μ\muμ is absolute rate (e.g., +$5/year)

  • Geometric BM: dX=μX dt+σX dBdX = \mu X \, dt + \sigma X \, dBdX=μXdt+σXdB → μ\muμ is relative rate (e.g., +5%/year)

Drift coefficient μ\muμ versus diffusion coefficient σ\sigmaσ:

  • μ\muμ has units [value/time] (or [1/time] if relative)

  • σ\sigmaσ has units [value/time\sqrt{\text{time}}time​] (or [1/time\sqrt{\text{time}}time​] if relative)

  • Over time Δt\Delta tΔt: drift contribution is O(Δt)O(\Delta t)O(Δt), diffusion is O(Δt)O(\sqrt{\Delta t})O(Δt​)

Discrete approximation (for intuition):

X(t+Δt)−X(t)≈μ(X,t)Δt+σ(X,t)Δt⋅ZX(t + \Delta t) - X(t) \approx \mu(X, t)\Delta t + \sigma(X, t)\sqrt{\Delta t} \cdot ZX(t+Δt)−X(t)≈μ(X,t)Δt+σ(X,t)Δt​⋅Z

where Z∼N(0,1)Z \sim \mathcal{N}(0, 1)Z∼N(0,1).

  1. Integration: Two Interpretations

For the integral ∫σ(X)dB\int \sigma(X)dB∫σ(X)dB, there are two conventions:

Itô integral (most common):

  • σ\sigmaσ evaluated at the left endpoint of each interval

  • Results in a martingale (E[X(t)∣X(s)]=X(s)\mathbb{E}[X(t) | X(s)] = X(s)E[X(t)∣X(s)]=X(s))

  • Non-anticipating (causal)

Stratonovich integral:

  • σ\sigmaσ evaluated at the midpoint

  • Obeys ordinary chain rule

  • Written as ∫σ(X)∘dB\int \sigma(X) \circ dB∫σ(X)∘dB

For computer science applications (simulations, filtering), Itô is standard.

  1. Computational Perspective

Simulating dX=μdt+σdBdX = \mu dt + \sigma dBdX=μdt+σdB from time ttt to t+Δtt + \Delta tt+Δt:

X_new = X_old + mu * Delta_t + sigma * sqrt(Delta_t) * randn()

This is the Euler-Maruyama method, the stochastic analog of Euler's method.

  1. dBdBdB is not a traditional differential – it's an infinitesimal random variable
